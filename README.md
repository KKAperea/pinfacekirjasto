Описание программы
Данная программа предназначена для обработки видеопотока с целью обнаружения, распознавания и анализа лиц. Она использует библиотеку OpenCV для работы с изображениями и видеопотоками, а также дополнительные модули для работы с базой данных лиц и настройки параметров.

Основные функции
Инициализация и настройка:

Программа начинает с вывода сообщения о версии и инициализации необходимых модулей.

Используется объект PinFace для работы с лицами, который поддерживает различные режимы обнаружения (ffmode) и распознавания (frmode).

Загрузка данных:

Загружаются известные лица из базы данных (face_data).

Импортируются конфигурационные параметры из модуля config.

Обработка видеопотока:

Программа читает видеопоток и обрабатывает каждый кадр.

Для каждого кадра выполняется обнаружение лиц, их распознавание и анализ.

Распознавание лиц:

Для каждого обнаруженного лица вычисляется векторное представление.

Производится поиск в базе данных известных лиц и вычисление расстояния до ближайших совпадений.

Если лицо не распознано, оно сохраняется в папку unknown.

Отображение результатов:

На изображение накладывается информация о распознанных лицах, включая имя, процент совпадения и другие данные.

Результаты отображаются в окне OpenCV.

Логирование:

В консоль выводятся подробные данные о каждом обработанном кадре, включая время выполнения операций и результаты распознавания.

Используемые технологии
OpenCV: Для работы с изображениями и видеопотоками.

NumPy: Для работы с массивами данных.

База данных лиц: Для хранения и поиска известных лиц.

Конфигурационные файлы: Для настройки параметров программы.

Как использовать
Установка зависимостей:
Убедитесь, что установлены все необходимые библиотеки:



pip install opencv-python numpy
Настройка конфигурации:

Отредактируйте файл config.py, чтобы указать параметры видеопотока, пути к данным и другие настройки.

Запуск программы:

Запустите скрипт:


python main.py
Программа начнет обработку видеопотока и отобразит результаты в окне OpenCV.

Управление:

Для выхода из программы нажмите клавишу q.
